# aspectsoftheday.js

Stores all Aspect of the Day winners.

Examples:

- Daddy of the Day
- Princess of the Day
- Animal of the Day

## Add New Aspect

Add to:

```js
const aspectsOfTheDay = {
  wizard: {}
};
```

Add trigger:

```js
const aspectsOfTheDayTriggers = {
  wizard: 100
};
```

Add alias:

```js
wizardofday: "wizard"
```

Add no winner message:

```js
wizard: "There is no Wizard of the Day yet!"
```
